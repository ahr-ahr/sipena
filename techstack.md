# Tech Stack

Backend : Laravel 12
Runtime : Laravel Octane (RoadRunner)
Web Server : Nginx
Frontend : Blade + Alpine.js
CSS : Tailwind CSS
Database : MySQL / MariaDB
Real-time : Laravel Reverb
DB Admin : phpMyAdmin
Cache/Session : Redis
Queue : Redis
Auth : Laravel Breeze (custom)
Container : Docker
Orchestration : Docker Compose
Scale-up : Kubernetes
Filesystem disk: Minio

Monitoring:

- Health Check Endpoint
- Docker stats / cAdvisor
- Prometheus + Grafana
- Laravel Pulse (opsional)
